# 速览模式 — Speed Read

## 目标

快速判断一本书讲什么、适合谁、值不值得读，以及应该怎么读。

输出到对话，并保存为：

```text
book-notes/<name>/speed-read.md
```

## 输出结构

```markdown
# <书名> — 速览

> **书名**：...
>
> **作者**：...
>
> **类型**：思想 / 技术 / 商业 / 历史 / 文学 / 传记 / 其他
>
> **适合人群**：...
>
> **阅读难度**：⭐ 到 ⭐⭐⭐⭐⭐
>
> **一句话总结**：...

## 这本书解决什么问题

## 核心观点

1. ...
2. ...
3. ...

## 全书结构

按章节或部分说明每一块在全书中的作用。

## 最值得读的部分

## 可能不适合你的地方

## 建议阅读路线

## 三个预判问题

**Q1：...**

**Q2：...**

**Q3：...**
```

## 要求

- 500-900 中文字。
- 不逐章机械摘要，先提炼全书问题链。
- 如果书中信息不足，写“材料中未说明”。
- 完成后提示用户可以切换到精读、伴读或分享模式。
